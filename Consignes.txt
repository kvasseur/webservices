		API REST - Mediatheque
		     Web Services
---- METHODES
[Get] /books - donne la liste des livres
[Get] /books/[int] - donne le livre à l'identifiant donné
[Post] /books - crée un livre dans la base
[Delete] /books/[int] - supprime un livre dans la base
[Put] /books/[int] - met à jour un livre dans la base

---- TABLE
[    Book    ]
[ -id        ]
[ -name      ]
[ -category  ]
